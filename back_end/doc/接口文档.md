# 用户接口("/user")

## 登录

数据例

请求

```json
成功例
{
    "account" : "2024060023",
    "password": "265823"
}

失败例
{
    "account" : "202406023",
    "password": "265823"
}
```

返回

```json
{
    "status": 200,
    "data": {
        "id": 23,
        "token": "",
        "code": 200
    },
    "msg": "登录成功",
    "error": ""
}

{
    "status": 200,
    "data": "账号或密码错误",
    "msg": "ok",
    "error": ""
}
```

唯一不需要传输(token)的接口，其他都要鉴权

其他默认都要传输token

```json
{
    ...
	"token": "asdakjsdasoldkjlk12j31290s0-a9dasd123m"	
}
```

(1)请求方式:Get

(2)请求地址:"/login"

(3)请求参数(body):

|  变量名  | 是否必填 | 数据类型 |    数据例    |           备注            |
| :------: | :------: | :------: | :----------: | :-----------------------: |
|    id    |    是    |  String  |  “1231123”   |         员工账号          |
| password |    是    |  String  | "myPassword" | 长度不小于6位，不大于16位 |

(4)返回参数:

|  字段   |  说明  |     备注      |
| :-----: | :----: | :-----------: |
| status  | 状态码 | 200：请求成功 |
|  data   | token  |   json格式    |
| message |  信息  | 成功/错误信息 |

## 注册

请求

```json
{
    "employee_list":[
        {
            "name" : "123",
            "tel" : "1123",
            "salary" : 123.33,
            "note" : "1231",
            "level" : 1231
        },
        {
            "name" : "123",
            "tel" : "1265823",
            "salary" : 123.33,

        }

```

返回

```json
{
    "status": 200,
    "data": {
        "success_list": [
            {
                "id": 23,
                "account": "2024060023",
                "password": "265823",
                "name": "123",
                "tel": "1265823",
                "salary": 123,
                "note": "1231",
                "level": 1231
            }
        ],
        "fail_list": [
            {
                "id": 0,
                "name": "123",
                "tel": "1123",
                "salary": 123.33,
                "note": "1231",
                "level": 1231,

        "code": 200

    "msg": "ok",
    "error": ""
```

(1)请求方式:Post

(2)请求地址:"/signup"

(3)请求参数(body):

|  变量名  | 是否必填 | 数据类型 |    数据例     |           备注            |
| :------: | :------: | :------: | :-----------: | :-----------------------: |

| password |    是    |  String  | "myPassword"  | 长度不小于6位，不大于16位 |
|   tel    |    是    |  String  | ”13933223456“ |         联系电话          |
|  salary  |    是    |  double  |    1234.45    |       工资，浮点数        |
